# Guess The Number

A number-guessing game with a custom dark-themed GUI, built with Python and Tkinter.

## Features
- Random number generation (1–100)
- On-screen numeric keypad (no physical keyboard needed)
- Live attempt counter and best-score tracking
- Instant higher/lower feedback with color-coded messages
- Reset button to start a new game anytime

## Tech Used
- Python 3
- Tkinter (GUI)

## How to Run
```bash
python guess_the_number.py
